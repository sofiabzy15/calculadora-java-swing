# calculadora-java-swing
# 🧮 Java Swing Calculator with ArrayList History

### 🇧🇷 Português
Calculadora funcional com interface gráfica desenvolvida para praticar os fundamentos de Java.
* **Destaque técnico:** Uso de **ArrayList** para criar um histórico dinâmico de operações que persiste durante a execução.
* **Interface:** Criada com a biblioteca nativa **Swing**.

### 🇺🇸 English
Functional calculator with a graphical user interface (GUI) developed to practice Java fundamentals.
* **Technical highlight:** Use of **ArrayList** to create a dynamic operation history that persists during execution.
* **Interface:** Built with the native **Swing** library.
